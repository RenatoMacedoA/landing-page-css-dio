# 🚀 Landing Page - Desafio DIO

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio da **Digital Innovation One (DIO)**, com o objetivo de praticar e aprimorar habilidades em desenvolvimento front-end.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivos

* Criar uma landing page moderna
* Aplicar conceitos de layout e responsividade
* Melhorar a organização do código

---

## 💻 Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```
cd seu-repositorio
```

3. Abra o arquivo:

```
index.html
```
